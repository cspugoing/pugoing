# 窗帘电机

#### 窗帘电机对接系统：

a) 语音命令“小乔管家，发现设备”

b) 待主机回应”已打开发现设备功能“

c) 捅一下学习码孔

d) 主机会语音提示“发现新设备，窗帘”

e) 此时进入小程序界面会看到设备栏新增加了窗帘的图标



#### 遥控器对接窗帘电机（适配遥控器）：

a) 长按电机学习码，5秒后电机指示灯变成长亮，然后松手此时表示成功进入了学码状态

 b) 电机在学码状态下按一下遥控器上键，指示灯闪三次表示学码成功



#### 窗帘解绑：

a) 长捅窗帘电机学习码孔，5s后电机指示灯长亮，松手此时进入学码状态

b) 然后再长捅一下学习码孔直到指示灯闪烁三次表示清码成功，解绑成功且窗帘恢复出厂设置
